== Installation ==

This section describes how to install the plugin and get it working.

1. Upload "instagram-followers-shortcode.zip" to the "/wp-content/plugins/" directory and unzip the file or you can install it 
	from "Plugins>>Add New" Page or went here :: "http://www.YourDomain.com/wp-admin/plugin-install.php"
2. Activate the plugin through the 'Plugins' menu in WordPress

3. Setting up the shortcode:
	
	[instagramFollowers id="25025320" limit="20"]
	Here "id" & "limit" are the attributes - some more available attributes with description are written below.
Attributes :
	
	i) id : go here "http://jelled.com/instagram/lookup-user-id" - give your instagram username and than 
	click submit button. Copy the user id from the page.
	ii) limit : No of followers you want to show up on the page. max followers limit are 50.
	iii) width : Integer value in pixels.
	iv) supportAuthor : "true" or "false" .As you can understand we are giving away some awesome wordpress plugins in free - so we must need your support. Rest of your wish :)
